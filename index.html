Оценка по формуле

Итог = (сумма баллов + 2) / 3, но не более 10. Самооценка по каждой задаче.

Как формируется оценка

  • 1 балл — задача с одной звёздочкой (★) — типы №1–10
  • 2 балла — задача с двумя звёздочками (★★) — типы №11–16
  • 3 балла — задача с тремя звёздочками (★★★) — типы №17–20

Максимум за вариант: 1·10 + 2·6 + 3·4 = 34 балла. Итог: min(10, (балл + 2) / 3).

Правила экзамена

  1. На экзамене не разрешается пользоваться никакой дополнительной информацией: чит-листы и телефоны запрещены.
  2. За каждый правильный шаг начисляются баллы. Лучше расписывать решение поэтапно. Если ответ недостаточно обоснован, оценка снижается.
  3. Если используешь теорему или лемму — напиши какую.
  4. Если решение неверное — зачеркни и напиши новое. Оценивается только первое.
  5. Можно завершить работу досрочно.

Как вводить ответы

В этом тренажёре все задачи — с развёрнутым решением. После завершения варианта тебе будет показано подробное решение каждой задачи и критерии оценивания. Ты сам ставишь себе балл от 0 до максимума за каждую задачу.

Поле для черновика автоматически сохраняется. Можно отметить задачу как решённую, нажав кнопку «Отметить решённой».
