# РешуМИА · Тренажёр к экзамену по «Математике в действии»

[![Deployed on GitHub Pages](https://img.shields.io/badge/GitHub%20Pages-online-black?logo=github)](https://xwooshiel.github.io/reshu-mia/)
[![Made with vanilla JS](https://img.shields.io/badge/JS-vanilla-yellow?logo=javascript)](#)
[![KaTeX](https://img.shields.io/badge/math-KaTeX-329)](https://katex.org/)

**[→ Открыть тренажёр](https://xwooshiel.github.io/reshu-mia/)**

Статический сайт-тренажёр для подготовки к экзамену по курсу «Математика в действии». Сделан в стиле «Решу ЕГЭ»: можно сгенерировать случайный вариант (по одной задаче из каждого из 20 типов) или прорешать каждый тип отдельно.

## Что внутри

- **20 типов задач** — векторы, матрицы, частные производные, цепочки, интегралы, оптимизация и т. д.
- **146 реальных задач** из летнего экзамена 2025, контрольных работ 2024–2025 и контестов курса.
- **Подробные решения** к каждой задаче, формулы через KaTeX.
- **Таймер на 180 минут**, автосохранение прогресса в `localStorage`.
- **Самооценка** по каждой задаче и итоговая оценка по формуле `min(10, (баллы + 2) / 3)`.

## Возможности

| Что | Как работает |
|---|---|
| Сгенерировать вариант | На главной нажми **«Сгенерировать вариант»**. Случайно берётся одна задача из каждого типа. Дальше — таймер 180 минут, навигация по 20 номерам, кнопка «Отметить решённой» и автосохранение черновика. |
| Прорешать по типам | На главной нажми **«Прорешать по типам»**. 20 карточек с типами, внутри — все задачи этого типа из всех источников. Слева список — справа условие, поле для черновика и кнопка показать разбор. |
| Завершить досрочно | В шапке экзамена есть кнопка **«Завершить»**. После подтверждения — экран результатов с решениями, критериями и самооценкой. |
| Восстановление | При перезагрузке страницы или закрытии браузера — состояние варианта (черновики, отметки «решено», таймер) автоматически восстанавливается. |

## Структура

```
site/
├── index.html              ← основная страница (SPA)
├── css/
│   └── styles.css          ← минималистичные чёрно-белые стили
└── js/
    ├── app.js              ← SPA-логика (роутинг, таймер, оценка, localStorage)
    ├── tasks.js            ← скелет банка (заголовки и критерии 20 типов)
    ├── tasks-part1.js      ← задачи типов 1–4
    ├── tasks-part2.js      ← задачи типов 5–8
    ├── tasks-part3.js      ← задачи типов 9–12
    ├── tasks-part4.js      ← задачи типов 13–16
    └── tasks-part5.js      ← задачи типов 17–20
```

## Запуск локально

```bash
cd site
python -m http.server 8765
# или
npx --yes serve .
```

Затем открой <http://localhost:8765>.

## Технологии

- Чистый ES2017+ JavaScript, без сборщиков и зависимостей.
- [KaTeX](https://katex.org/) для рендера формул.
- Шрифты: [Inter](https://rsms.me/inter/) + [JetBrains Mono](https://www.jetbrains.com/lp/mono/).
- `localStorage` для автосохранения.
